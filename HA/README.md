# Motorradhelm-Suche (MVC-Pattern)

Dieses Projekt implementiert eine Helm-Suche im MVC-Pattern. Das Model ist vorgegeben, die Logik und View müssen implementiert werden.

## 📌 Projektstruktur

```
📂 HelmWaWi
├── 📂 Model
│   ├── __init__.py
│   ├── Helm.py
│   ├── HelmBestand.py
├── 📂 Controller
│   ├── __init__.py
│   ├── Logic.py
├── 📂 View
│   ├── __init__.py
│   ├── Terminal.py
├── main.py
└── README.md
```

## ▶️ Programm starten

```sh
python main.py
```
